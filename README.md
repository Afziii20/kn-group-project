# KN Group - AI Powered Storefront 🚀

A premium e-commerce platform for modern Indian appliances.

## 🛠 Tech Stack
- **Frontend:** React, Tailwind CSS, Framer Motion
- **Backend:** FastAPI (Python), PostgreSQL, SQLAlchemy
- **AI Integration:** Google Gemini 2.0 Flash

## 🚀 Getting Started
1. Clone the repo: `git clone https://github.com/Afziii20/kn-group-project.git`
2. Set up Backend: `cd kngroup-backend && pip install -r requirements.txt`
3. Set up Frontend: `cd kngroup-frontend && npm install`
